图片和视频建议填写外链 URL。不要把大视频直接转成 Base64 存进 JSON，否则 GitHub 和 Vercel 都会变慢。

Vercel 环境变量

ADMIN_USER=你的后台账号 ADMIN_PASSWORD=你的后台密码 GITHUB_TOKEN=GitHub fine-grained token，允许 Contents: Read and write GITHUB_OWNER=foulmora GITHUB_REPO=AIGosling GITHUB_BRANCH=main CONTENT_PATH=content.json
发布按钮会调用 /api/content，把后台编辑的内容提交到 GitHub。Vercel 连接该仓库后会自动重新部署。
